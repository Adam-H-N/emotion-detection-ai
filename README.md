# emotion-detection-ai
AI-based emotion detection using OpenCV and deep learning
# 🎭 Real-Time Emotion Detection AI

## 📌 Overview
This project detects human emotions in real-time using a webcam.  
It uses computer vision and deep learning to analyze facial expressions and display emotions with confidence scores.

## 🚀 Features
- Real-time face emotion detection
- Emotion smoothing using history buffer
- GUI interface using Tkinter
- Voice feedback using text-to-speech
- Emotion probability bars

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- FER (Facial Emotion Recognition)
- Tkinter
- pyttsx3

## ▶️ How to Run

1. Clone the repository: git clone https://github.com/Adam-H-N/emotion-detection-ai.git
2. cd emotion-detection-ai
3. Install dependencies: pip install -r requirements.txt
4. Run the program: python main.py
## 📁 Project Structure
- `main.py` → Main application
- `requirements.txt` → Dependencies
- `assets/` → Images/screenshots
